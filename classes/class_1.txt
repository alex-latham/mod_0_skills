Class: BreakfastBurrito

Attributes:
name (string)
price (float)
fillings (array)
smothered (boolean)
size (string)

Methods:
put_on_sale (reduces the price float value by 10% and appends " - SALE!" to the end of the name string)
desmother (changes the smothered boolean value to false)
make_vegetarian (removes all elements containing "sausage" or "bacon" from the fillings array)
super_size (changes the size string value to "XXXL")
